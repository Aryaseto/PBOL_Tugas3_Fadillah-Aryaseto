* Fitur didalam websitenya:
- Menggunakan bahasa pemrograman HTML dengan gabungkan jsp.
- Menggunakan CSS seperti margin, padding, border, dan backgroud.
- Serta ada tambahan dari JavaScript untuk menyimpan data mahasiswanya.
- Tambahan menggunakan looping untuk menampilkan semua data yang dikirim dari forms.
- Method getParameter() untuk mengambil setiap nilai inputan.
- Method getParameterValues() untuk data yang memiliki banyak nilainya.
